# Excel-Advanced-Analytics-Coursework
A comprehensive collection of applied data analytics, statistical modeling, and data visualization workbooks developed across three academic coursework practicals in Microsoft Excel.

 Comprehensive Skill Sheet :
1. Advanced Core & Text Functions:
   Logical Formulas: Evaluated multi-criteria scenarios using complex logical statements (`NESTED IF`, `AND`, `OR`, `NOT`).
   Summary Functions: Implemented dynamic criteria metrics through conditional counting and summing (`COUNT`, `COUNTA`, `COUNTBLANK`, `COUNTIF`, `SUMIF`).
Data Cleansing: Standardized text datasets using casing adjustments and spacing remediation (`PROPER`, `TRIM`, `UPPER`, `LOWER`).
Lookup & Reference: Applied data retrieval arrays using index-matching and lookup configurations.

2. Applied Descriptive Statistics
Central Tendency Modeling: Engineered manual calculation templates for finding the Mean, Median, and Mode from grouped frequency data distributions.
Dispersion Analysis: Constructed formula pathways to calculate variance boundaries and Standard Deviation (SD).
Distribution Overviews: Formulated demographic splits (e.g., population/gender finish metrics) via full 5-Point Summaries (Minimum, Quartile 1, Median, Quartile 3, Maximum).

3. Economic Data Modeling & Advanced Charts
Macroeconomic Analytics: Modeled overall wealth distribution inequalities by mapping an empirical Lorenz Curve.
Area Calculation:Developed calculation frameworks to compute the definitive area beneath the Lorenz curve to output the exact Gini Coefficient.

4. Professional Data Visualization
Categorical Visuals: Selected appropriate graphical structures across qualitative metrics (Bar Charts, Pie Charts, Area Layouts, Line Graphs).
Statistical Charts:Generated frequency distribution Histograms paired with Pareto Charts to highlight dominant operational elements.
Advanced Dashboards: Implemented comparative Scatter Diagrams, localized conditional Heat Maps, and micro-trend inline Sparklines.

Focus: Implementation of executive data assets, multi-series category charts, Pareto analysis, heat maps, and localized cell-embedded sparklines.
